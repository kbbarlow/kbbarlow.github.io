CSS Box Model

In the box model, there are 3 different parts. They are padding, border, and margin.

Padding occurs inside of the background of the element. Padding is used to make space from itself

Borders are the line of the box

Margin is the space around the outside of the box. In general, margin is used to put space between two elements.


This box is 200px wide x 200px high, px are an absoulte value. This value wont change not matter what else changes

This box is 20vw wide x 20vh high, 20 vw and vh is 20% of the viewport. This value is relative to the viewport

This box is 20em wide x 20em high. em is relative to the parent's font size. The .wrapper parent is set to 1em. You have to pay careful attention to the parents size when using em.

This box is 20rem wide x 20rem high. rem stands for root em. rem is relative to the root font size. Default font size is 16px. Notice the em box and rem box are the same. On this page 1em = 16px
